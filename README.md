Excited to share my latest project: an Arduino Car that combines automation, control, and creativity into a single build! Here's what makes this project stand out: 

### Components Used: 
- Arduino UNO: The brain behind the car. 
- HC-SR04 Ultrasonic Sensor & Servo Motor: For obstacle detection and navigation. 
- HC05 Bluetooth Module: Enables wireless control via the mobile app 'RoboBoy.' 
- DHT Sensor: Monitors and displays temperature & humidity. 
- L298 Motor Driver: Powers the car's movement. 
- LED Lights: Functional front and rear lights. 
- LCD Display with I2C: Acts as the number plate and displays sensor data. 
- Buzzer Alarm: For the horn. 
- 12V Battery, Jumper Wires, Chassis, and Wheels: To bring it all together!
